4. re.match()返回值问题

re.match()匹配成功返回re.match()对象，可用.group()方法提取字符串；匹配失败返回None。

文件时是按行读取的，因此循环对行进行正则匹配，返回的结果中穿插着None和re.match对象，而None调用.group()方法会报错，所以在这简单的用if过滤掉了None。
